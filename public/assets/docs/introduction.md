# Ant Design Pro 介绍

Ant Design Pro 是一个企业级中后台前端/设计解决方案，基于 Ant Design 体系精心打造。它提供了丰富的组件和模板，帮助开发者快速构建高质量的企业级应用。

## 设计理念

Ant Design Pro 致力于在设计规范和基础组件的基础上，继续向上构建，提炼出典型模板/业务组件/配套设计资源，进一步提升企业级中后台产品设计研发过程中的用户和设计者的体验。

### 设计价值观

1. **自然** - 自然的交互，自然的反馈
2. **确定性** - 设计清晰明确，操作路径明确
3. **意义感** - 界面是富有生命力的
4. **生长性** - 设计系统是可生长的

## 主要特性

### 丰富的组件

Ant Design Pro 提供超过 50 个高质量组件，覆盖中后台系统的各种场景：

- **布局组件**：高级布局、网格系统、卡片布局
- **数据展示**：表格、列表、卡片、时间轴
- **数据录入**：表单、输入框、选择器、上传
- **导航组件**：菜单、面包屑、分页、标签页
- **反馈组件**：模态框、通知、加载状态、进度条
